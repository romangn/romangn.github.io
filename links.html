This page is a collection of various programming resources that I found to be interesting and useful. Includes links to other online resources and some recommended books.

Online resources

  • Leetcode

    • A great platform to get better at alroithms and data structures. Contains a large number of questions and supports a variety of programming languages.

  • Visualgo

    • Visual representation of operations related to algorithms and data structures. Highly useful for better understanding of data structures.

  • Big O cheat sheet

    • A website that shows Big O complexity for various CS algorithms and data structures operations.

  • SDL 2 Tutorial

    • A good starting tutorial that allows to quickly get familiar with SDL and its usage with C++.

  • Pluralsight

    • A great learning resource that provies a huge collection of video courses on a large number of programming topics, especially the ones related to .NET. Requires subscription but free 6 months subscription can be gained by creating a visual studio account.

  • ASP.NET Core Documentation

    • A good starting resource for the latest version of ASP.NET. Recommended for both people new to ASP NET and people who worked with previous versions of ASP.NET.

Books

  • C++ Primer 5th Edition

    • A great book on C++ and 11 standard. Contains a lot of essential information on C++ and STL. In addition to this, the book also contains a large number of programming excercises which enable the reader to get bettet understanding of the topic. Not recommended to complete beginners to programming since it assumes the reader already knows programming.

  • Invent with Python

    • A collection of Python books which are good for complete beginners to programming. The author explaines the information very clearly and programming excercises are a lot of fun. The books can also be read online for free.

  • Unity in Action

    • A book on unity which I found quite useful. The personal advantage of this book for me is that it is aimed specifically at programmers in comparison to other resources which are aimed at designers/artists.
